Abstract

O principal objetivo deste TPC era criar uma aplicação ToDo List single page. Nesta página conseguimos encontrar as tarefas que ainda faltam realizar, bem como aquelas que já foram realizadas.

Cada tarefa é constituída por um identificador único (id), um nome (name), uma breve descrição (description), uma data de ínicio e de fim (register_date e deadline) e por último possui um tipo, indicando se a tarefa já foi feita (done) ou ainda falta fazer (todo).

Na página principal encontra-se um botão de adicionar task, que permite criar novas tasks para realizar. Assim criada uma tarefa, caso o utilizador pretenda editar a mesma, basta clicar no id da tarefa. Para as tarefas já realizadas é possível elimina-las, para isso basta clicar na x no fim da linha da tarefa.
