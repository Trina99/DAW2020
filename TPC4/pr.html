Abstract

Este tpc foi constituido por várias etapas. A primeira consistiu em criar vários ficheiros html com nomes específicos(i.e. arq1.html, arq2.html,...) a partir de um database (arq.xml). Para tal, foi usado o XSLT para transformar o ficheiro de dados XML.

De seguida, foi criado um servidor localhost, em nodejs, para responder a pedidos. O indice criado pede ao servidor para aceder às paginas, em vez de as ir buscar aos ficheiros do computador.

Por último foi realizado o presente project record com o modelo definido na aula.
